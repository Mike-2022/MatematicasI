El tema de sucesiones y series tiene una gran aplicación en áreas como ciencias de la computación, ingeniería, física, economía y finanzas, entre otras. Asimismo, este tema es la base para deducir varias fórmulas utilizadas en matemáticas financieras.


SucesionesUna sucesión es un conjunto de números en un cierto orden. El orden lo proporciona una regla, la cual nos dice como calcular el valor de cada termino, dicha regla es una fórmula.

Cada objeto o número de la sucesión recibe el nombre de término.



Ejemplos:
3, 5, 7, 9, 11, 13, 15... Es una sucesión porque están en orden y siguen la regla (2n + 1)
1, 4, 9, 16, 25, 36…    Es una sucesión porque están en orden y siguen la regla (n2)
1, 3, 4, 0, 7, 15…        No es una sucesión porque no están en orden y no siguen ninguna regla.

Cuando una sucesión tiene un número fijo de términos decimos que es una sucesión finita.

Cuando una sucesión tiene un número infinito de términos decimos que es una sucesión infinita, indicada por los tres puntos suspensivos (...) la cual se llama elipsis e indican que los términos siguientes tienen el mismo patrón que el establecido por los ya dados.

Ejemplos:
1, 2, 3, 4 , ...         Sucesión infinita, indicado por los puntos suspensivos
20, 25, 30, 35, ...  También es una sucesión infinita.
1, 3, 5, 7               Sucesión finita, contiene los 4 primeros números impares
4, 3, 2, 1               Sucesión finita y va de 4 a 1.
1, 2, 4, 8, 16, ...    Sucesión infinita donde vamos duplicando cada término.
a, b, c, d, e           Sucesión finita formada por las 5 primeras letras en orden alfabético
0, 1, 0, 1, ...          Sucesión infinita que alterna 0s y 1s (siguen un orden alternativo) 

El conjunto 8, 13, 18, 23, 28, ... es una sucesión de números cuya regla es que cada término, después del primero, se obtiene sumando 5 al término anterior.

El conjunto 4, 8, 16, 32, 64, ... es una sucesión cuya regla es que cada término, después del primero, se obtiene multiplicando por 2 el término anterior.

Si se representa con a1 al primer término de una sucesión, con a2 al segundo término, con a3 al tercer término, y así sucesivamente, entonces se puede escribir la sucesión como: 

\(a_1, a_2, a_3, a_4, {...}, a_n, ...\)

El término an se llama término n-ésimo y se encuentra en la posición número n.

Como a cada número natural n le corresponde un número an, una sucesión se define de la siguiente forma:

Una sucesión es una función cuyo dominio es el conjunto de los números enteros positivos.

Para representar cada elemento de una sucesión usamos la sintaxis general:

\(a_n = Regla\)

Donde:
an es un término cualquiera de la sucesión y se lee el termino enésimo.
n es la posición del elemento, toma valores enteros positivos.
Regla es la fórmula que nos permite calcular cada elemento.

Veamos la sucesión 3, 5, 7, 9, … donde los valores de los términos se obtienen con la regla \(2n + 1\).

Esta sucesión la representamos como: 

\(\displaystyle {a}_{n}=2n+1 \)

Para calcular cualquier término de la sucesión, solo basta con sustituir n por la posición del término deseado.

Primer término n = 1           \(\displaystyle {a}_{1}=2(1)+1=3 \)
Segundo término n = 2       \(\displaystyle {a}_{2}=2(2)+1=5 \)
Tercer término n = 3            \(\displaystyle {a}_{3}=2(3)+1=7 \)
Decimo término n = 10       \(\displaystyle {a}_{10}=2(10)+1=21 \) 

Digamos que tenemos una sucesión representada por la ecuación:

\(\displaystyle {a}_{n}=n^2 \)

Calculemos los tres primeros términos

Primer término n = 1             \(\displaystyle {a}_{1}=1^2=1 \)
Segundo término n = 2         \(\displaystyle {a}_{2}=2^2=4 \)
Tercer término n = 3              \(\displaystyle {a}_{3}=3^2=9 \) 

Así, la fórmula \(a_n = 3n - 2\) define una sucesión cuyos primeros seis términos son:

\(a_1 = 3(1) - 2 = 1\)
\(a_2 = 3(2) - 2 = 4\)
\(a_3 = 3(3) - 2 = 7\)
\(a_4 = 3(4) - 2 = 10\)
\(a_5 = 3(5) - 2 = 13\)
\(a_6 = 3(6) - 2 = 16\)

Entonces, la sucesión definida por la fórmula \(a_n = 3n − 2\), es 1, 4, 7, 10, 13, 16, …

Si se desea calcular el vigésimo sexto término de esta sucesión, entonces se usa n = 26
 
\(a_{26} = 3(26) - 2 = 76\)

En matemáticas existen diferentes tipos de sucesiones: Aritméticas, Geométricas, Triangulares, Fibonacci, etc. Para nuestro estudio solo veremos las dos primeras. 

Ejemplo. Enumere los primeros cinco términos y el vigésimo término de la sucesión definida mediante la fórmula

\(\displaystyle a_{n}=\frac{n-2}{n} \)

Para n = 1, entonces \(a_{1}=\frac{1-2}{1}=-1 \)

Para n = 2, entonces \(a_{2}=\frac{2-2}{2}=0 \)

Para n = 3, entonces \(a_{3}=\frac{3-2}{3}=\frac{1}{3} \)

Para n = 4, entonces \(a_{4}=\frac{4-2}{4}=\frac{2}{4}=\frac{1}{2} \)

Para n = 5, entonces \(a_{5}=\frac{5-2}{5}=\frac{3}{5} \)

Para n = 20, entonces \(a_{20}=\frac{20-2}{20}=\frac{18}{20}=\frac{9}{10} \)

La sucesión se escribe como -1, 0, 1/3, 1/2, 3/5, ..., 9/10, ...

Otra forma de definir una sucesión es mediante una fórmula recursiva. Ésta consiste en que se dan los valores de uno o varios términos de la sucesión y el término n-ésimo, an, se expresa en función de los términos precedentes, como se muestra en el siguiente ejemplo.

Ejemplo. Escriba los primeros cinco términos de la sucesión definida recursivamente como

\(\displaystyle a_{n}=4a_{(n-1)}-10,\)

con \(a_{1}=8 \)

El primer término se conoce, ya que es dato del problema, y es a1 = 8.

Para el segundo término, se sustituye n = 2 en la fórmula recursiva y se obtiene:

\(\displaystyle a_{2}=4a_{(2-1)}-10=4a_{1}-10=4(8)-10=22 \)

El tercer término se obtiene al sustituir n = 3 en la fórmula

\(\displaystyle a_{3}=4a_{(3-1)}-10=4a_{2}-10=4(22)-10=78 \)

Para el cuarto término n = 4

\(\displaystyle a_{4}=4a_{(4-1)}-10=4a_{3}-10=4(78)-10=302 \)

Para el quinto término n = 5

\(\displaystyle a_{5}=4a_{(5-1)}-10=4a_{4}-10=4(302)-10=1,198 \)

Como se puede observar, para obtener un término cualquiera se requiere conocer el valor del término precedente.

La sucesión se escribe como 8, 22, 78, 302, 1,198, ...



SeriesUna serie llamada también progresión se define como la suma de todos los términos de una sucesión.

La expresión de una serie aritmética es: 

\(\displaystyle {S}_{n}={a}_{1}+{a}_{2}+{a}_{3}+...+{a}_{n} \)

Esta expresión se puede escribir de manera simplificada usando la notación sigma: 

\(\displaystyle {S}_{n}=\sum _{i=1}^{n}{a}_{i} \)

que se lee sumatoria de los términos ai para i =1 hasta n.

Así, para la sucesión 10, 15, 20, 25, 30, la serie es la suma de los primeros 5 términos de la sucesión \( {a}_{n} \):

\(\displaystyle {S}_{5}=\sum _{i=1}^{5}{a}_{i} ={a}_{1}+{a}_{2}+{a}_{3} +{a}_{4}+{a}_{5}\)

\(\displaystyle {S}_{5}=\sum _{i=1}^{5}{a}_{i} =10+15+20+25+30=100 \)

Una serie puede ser infinita o finita, según sea la sucesión infinita o finita.

Si n es un entero positivo, entonces \(S_n\) simboliza la suma de los primeros n términos de una sucesión. Así, para la sucesión \(a_1, a_2, a_3, a_4, ..., a_n, ...\), se tiene:

  \(S_1 = a_1\)                             S1 se llama primera suma parcial,

  \(S_2 = a_1 + a_2\)                    S2 es la segunda suma parcial,

  \(S_3 = a_1 + a_2 + a_3\)           S3 es la tercera suma parcial

  \(S_n = a_1 + a_2 + a_3 + a_4, + ..., + a_n\)        Sn es la n-ésima suma parcial.

La sucesión

\(S_1, S_2, S_3, S_4, ..., S_n, ...\)

se llama sucesión de sumas parciales.

Ejemplo. Calcule las primeras cuatro sumas parciales de la sucesión definida por

\(\displaystyle a_{n}={(-1)}^{(n+1)}{3}^{n} \)

Los primeros cuatro términos de la sucesión son:

\(\displaystyle a_{1}={(-1)}^{(1+1)}{3}^{1}={(-1)}^{2}{3}^{1}=3 \)

\(\displaystyle a_{2}={(-1)}^{(2+1)}{3}^{2}={(-1)}^{3}{3}^{2}=-9 \)

\(\displaystyle a_{3}={(-1)}^{(3+1)}{3}^{3}={(-1)}^{4}{3}^{3}=27 \)

\(\displaystyle a_{4}={(-1)}^{(4+1)}{3}^{4}={(-1)}^{5}{3}^{4}=-81 \)

Por lo tanto, las primeras cuatro sumas parciales son:

\(S_1 = a_1 = 3\)
\(S_2 = a_1 + a_2 = 3 - 9 = -6\)
\(S_3 = a_1 + a_2 + a_3 = 3 - 9 + 27 = 21\)
\(S_4 = a_1 + a_2 + a_3 + a_4 = 3 - 9 + 27 - 81 = -60\)

La sucesión de sumas parciales es: 3, -6, 21, -60
